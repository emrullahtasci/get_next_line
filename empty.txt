# 🟢 GetNextLine - 42 Kocaeli 🇹🇷

## 📝 Description
`get_next_line` fonksiyonu, bir dosyayı **satır satır okuyabilen** C fonksiyonudur.
- Her çağrı bir sonraki satırı döndürür.
- Dosya tek satır veya çok satırlı olabilir.
- Tüm **edge-case** durumları destekler: boş dosya, tek satır, son satırda newline yok, sadece newline, uzun satırlar.

🧠 Edge-case Behavior
Case	Expected Output
Empty file	No output (returns NULL)
Single-line file	Prints line, next call NULL
Last line without newline	Line still returned
Only newlines	Prints blank lines
Very long line	Entire line printed, even if longer than BUFFER_SIZE


📂 Example Usage (main.c)
#include <fcntl.h>
#include <unistd.h>
#include <stdio.h>
#include <stdlib.h>
#include "get_next_line.h"

int main(void)
{
    int fd = open("test_single_line.txt", O_RDONLY);
    char *line;

    if (fd < 0)
    {
        perror("open");
        return 1;
    }

    while ((line = get_next_line(fd)) != NULL)
    {
        printf("%s\n", line);
        free(line);
    }

    close(fd);
    return 0;
}
